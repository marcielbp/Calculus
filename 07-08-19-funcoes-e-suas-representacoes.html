Funções e suas representações

Uma função fff é uma lei que associa a cada elemento xxx pertencente a um domínio DDD, exatamente um elemento f(x)f(x)f(x) em um conjunto EEE.

Em geral consideramos as funções para qual DDD e EEE são conjuntos. DDD é denominado domínio da função fff, e EEE por sua vez representa o contradomínio.

O número f(x)f(x)f(x) para todo xxx que pertença a DDD, define um novo conjunto chamado de imagem, ou ImImIm. A imagem é constituída por todos os valores f(x)f(x)f(x) que são obtidos pela lei de formação de todos os valores de xxx no domínio.

Caso não seja possível aplicar a lei f(x)f(x)f(x) sobre algum valor de xxx, onde xxx seja real, diz-se que xxx não pertence ao domínio de fff.

Fundamentos de conjuntos

Elementos (xxx)

Valor único dentro de um conjunto.

Conjuntos

Agrupamento ou coleção de elementos, pode ser representado explicitamente por chaves {}\{\}{}.

Restrição de elementos (∣|∣)

Indicado pelo operador “tal que” (∣|∣), identifica condições para que o elemento xxx esteja no conjunto.

Pertinência e não-pertinência

Indica se um elemento xxx pertence ou não a um conjunto.

Conjuntos numéricos comuns

São os conjuntos mais utilizados para representar elementos no Cálculo.

Números inteiros (Z\ZZ)

Z={...,−3,−2,−1,0,1,2,3,...}\Z = \{..., -3, -2, -1, 0, 1, 2, 3,...\}Z={...,−3,−2,−1,0,1,2,3,...}

R={...,−2,...,−12,...,0,...,4,...,125}\R = \{..., -2, ..., -\frac{1}{2}, ..., 0, ..., 4, ..., \frac{12}{5}\}R={...,−2,...,−21​,...,0,...,4,...,512​}

"xxx pertence ao conjunto dos reais tal que xxx não é um número negativo, ou seja, xxx é maior ou igual a zero."

Representação gráfica de um função

O gráfico de uma função nos fornece uma imagem útil comportamental ou histórica da função. Para representa-lo utilizamos o sistema de coordenadas cartesianas, indicando os valores de xxx do domínio no eixo horizontal (abscisas) e confrontando-os com os valores obtidos após aplicar a lei de formação f(x)f(x)f(x), representando no eixo vertical (ordenadas).

Como encontrar o domínio e imagem da função?

1) Observar o gráfico pode indicar o comportamento da função e dar uma ideia sobre quais valores não fazem parte de seu domínio.

2) Testar os valores arbitrários de xxx na função afim de descobrir se existem valores que não podem fazer parte de DDD e ImImIm.
